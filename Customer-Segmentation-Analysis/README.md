# 🧠 Customer Segmentation Analysis

## 📌 Overview

This project uses K-Means clustering to segment customers based on their income and spending behavior. The goal is to identify meaningful customer groups that can help businesses improve marketing strategies and decision-making.

## 🎯 Objective

* Segment customers based on purchasing behavior
* Identify high-value customer groups
* Enable targeted marketing strategies

---

## 📊 Dataset

The dataset includes:

* Customer ID
* Age
* Annual Income (k$)
* Spending Score (1–100)

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

---

## 🔍 Project Workflow

### 1. Data Preprocessing

* Cleaned dataset
* Checked missing values
* Selected relevant features

### 2. Exploratory Data Analysis

* Visualized income vs spending
* Identified patterns and trends

### 3. K-Means Clustering

* Applied clustering algorithm
* Used Elbow Method for optimal clusters
* Segmented customers into groups

---

## 📈 Key Insights

* Customers form distinct segments based on income and spending
* High income does not always indicate high spending
* A key segment shows high spending with moderate income → ideal target customers
* Low-income, low-spending group contributes least

---

## 📊 Results

* Successfully created customer segments
* Each cluster represents unique behavior
* Useful for:

  * Targeted marketing
  * Customer retention
  * Business optimization

---

## 🚀 Conclusion

K-Means clustering effectively identifies hidden patterns in customer data. These insights can help businesses improve strategy and customer engagement.
