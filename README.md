### 🔹 2. Android App Market Analysis

📌 **Objective:** Analyze trends in the Android app market using exploratory data analysis.

🔧 **Tech Used:** Python, Pandas, Matplotlib, Seaborn
📊 **Highlights:**

* Data cleaning and preprocessing
* Category-wise app analysis

📌 **Objective:** Predict house prices using Linear Regression.

* Visualization of predictions
